## How to Make Images Show on GitHub

1. Create a folder named `images`
2. Add your screenshots inside it
3. Rename the screenshots exactly like this:

```bash
home.png
dashboard.png
students.png
teachers.png
attendance.png
fees.png
```

4. Commit and push the changes:

```bash
git add README.md images
git commit -m "Add project README with screenshots"
git push
```

## How to Run the Project Locally

```bash
npm install
npm run dev
```

Then open the local link shown in your terminal.
