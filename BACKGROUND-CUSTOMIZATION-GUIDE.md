## ğŸ†˜ Troubleshooting

### **Image Too Large:**
```
Error: "File too large! Choose under 5MB"
Solution:
- Compress the image
- Choose a smaller image
- Use online compressor
```

### **Image Won't Apply:**
```
Problem: Click apply but nothing happens
Solution:
- Check file is selected
- Check file size < 5MB
- Try different image
- Hard refresh (Ctrl + Shift + F5)
```

### **Storage Error:**
```
Error: "Image too large for storage!"
Solution:
- Browser localStorage is limited
- Use smaller image
- Clear old background first
- Try different browser
```

### **Lost Background After Refresh:**
```
Problem: Custom background disappears
Solution:
- Check browser isn't clearing data
- Disable "Clear on exit" in browser
- Reapply the background
```
